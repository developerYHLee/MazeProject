# MazeGame
### 미로를 외워서 탈출하는 C++ 콘솔 게임
머릿속의 지도와 감각으로 미로를 탈출하는 텍스트 형식의 콘솔 게임

### 🚀개발 동기
* 부트캠프 과제를 창의성 있고 완벽하게 수행하고 싶은 열망
  * 제주도 메이즈랜드의 경험에서 영감을 받음
* C++ 활용 역량 강화
<br></br>

### 📜게임 설명
* 설정한 크기에 맞는 미로 랜덤 생성
* 생성된 미로를 확인하고 게임 시작
* 텍스트 형식으로 미로 진행 및 탈출
<br></br>
